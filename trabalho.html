1ª Atividade - 22-10-2019

Criar uma página Html para representação de retas geradas pelos algoritmos DDA e Bresenham e para representação de círculos usando o algoritmo de círculo simples e o do ponto médio. As figuras serão desenhadas a partir de entradas o usuário. Para a reta, o usuário entra com os dois pontos. Para o círculo, a entrada é o raio e o centro do mesmo. Usar SVG para desenho das figuras. Podem ser usados recursos como JavaScript, CSS etc. para geração da Página. Equipes de até 3 componentes. Pode ser feito individual. Valor até 5 pontos na primeira nota.
